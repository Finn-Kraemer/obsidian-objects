# 💎 Welcome to the Objects Demo Vault

This vault is pre-configured to show you how **Objects** works.

## How to try it:

1.  **Open an Editor**: Open this file or create a new one.
2.  **Trigger**: Type `@` anywhere.
3.  **Select**: Choose an object type:
    -   `@project` (goes to `Projects/`)
    -   `@person` (goes to `People/`)
    -   `@atomic` (goes to `Zettelkasten/`)
4.  **Name it**: A modal will appear. Type a name like "Deep Work Project" or "John Doe".
5.  **Hit Enter**: Watch as a link is inserted and the new note is created and opened automatically.

## Pre-configured Settings:

-   **Template Folder**: `Templates/`
-   **Default Output**: `Inbox/`
-   **Triggers**:
    -   `@project` ⮕ `project.md` ⮕ `Projects/`
    -   `@person` ⮕ `person.md` ⮕ `People/`
    -   `@atomic` ⮕ `atomic.md` ⮕ `Zettelkasten/`
